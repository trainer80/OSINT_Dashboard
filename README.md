# 🔎 OSINT Dashboard

A Python-based Open Source Intelligence (OSINT) Dashboard built using Streamlit that aggregates domain, DNS, IP, and public news intelligence into a single interactive interface.

---

## 🚀 Features

- WHOIS Lookup (domain ownership details)
- DNS Enumeration (A, MX, NS records)
- IP Geolocation tracking
- Hacker News intelligence search
- Data visualization using Plotly
- Interactive Streamlit dashboard

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Requests
- Pandas
- Plotly
- python-whois
- dnspython
